User Guide

  • Draw: Click and drag on the canvas to draw.
  • Record: Click the button to start/stop recording your drawing.
  • Reproduce: Click the button to replay your drawing.
  • Undo/Redo: Use and to undo/redo strokes.
  • Eraser: Click to erase parts of your drawing.
  • Clear: Click to clear the canvas.
  • Color Picker: Select a color or use color history/common colors.
  • Brush Size: Adjust the brush size with the slider.
  • Save: Save your drawing as an image or JSON file.
  • Load: Load a drawing from a JSON file.
  • Dark Mode: Toggle dark mode with the button.

For more info, visit the GitHub page.
